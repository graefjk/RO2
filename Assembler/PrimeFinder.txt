//Assembler to find prime numbers
LOAD x4, 2
LOAD x5, 0
Label1
LOAD x2, 2
Label2
COMPARE x4, x2
//all numbers up to x4 have been checked and no divider has been found =>x4 is prime
JUMPZ prime 
LOAD x1, x4
CALL divider //divide x4 and x2
COMPARE x1, 0 // x2 divides x4 => x4 is not a prime
JUMPZ noPrime
ADD x2, 1
JUMP Label2

prime
STORE x4, x5
ADD x5, 1

noPrime
add x4, 1
JUMPC end //end the programm if all possible numbers have been tested
JUMP Label1


divider // div x1 and x2 saves divident in x3 and rest in x1
LOAD x3, 0 //rest
LoopStart
SUB x1, x2
JUMPC LoopEnd
ADD x3, 1
JUMP LoopStart
LoopEnd
ADD x1, x2
RETURN
